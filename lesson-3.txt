Câu 1: Giải thích các câu lệnh:
- git commit --amend: xoá mesage của file vừa commit
- git commit --amend-m"message": thay đổi nội dung message của file vừa commit
- git restore --staged <file>: lấy lại file đã chuyển sang vùng staging mà chưa commit
- git reset HEAD~1: lấy lại các file đã commit ở lần gần đây nhất
